# StoryMaker Platform Launchers

Generated by `corepack pnpm package:binaries`. The command name is historical; it writes launchers, not native single-file binaries.

These artifacts are deterministic Node-dependent platform launchers for the StoryMaker CLI compatibility entrypoint:

- `storyctl.exe`
- `storyctl-macos`
- `storyctl-linux`

The filenames keep `storyctl` for compatibility with existing StoryOS automation. They do not contain Node.js or the CLI source. They require Node.js 24 or newer on `PATH` and execute `packages/cli/dist/index.js`.
