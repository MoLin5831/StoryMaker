# 行为准则

StoryMaker 希望保持一个友善、具体、专业的协作环境。参与讨论、提交 issue、评审 PR 或维护文档时，请遵守以下原则：

- 对人友善，对问题具体。
- 解释事实和影响，避免人身攻击、嘲讽和骚扰。
- 尊重不同经验背景的贡献者。
- 讨论分歧时优先给出复现、证据和可执行建议。
- 不发布他人的隐私信息、凭据、真实用户数据或未授权内容。

不被接受的行为包括：骚扰、歧视性表达、威胁、持续打断讨论、公开羞辱、发布敏感信息，以及其他让协作者无法安全参与项目的行为。

维护者可以删除不当内容、关闭讨论、限制参与者互动，或在必要时采取进一步处理。严重安全或滥用问题请按 [安全政策](SECURITY.md) 报告。
